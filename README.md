# Bookish

A MERN stack application allowing users to create, host and join book clubs. Users may create a book club and invite other users to join as members. Books are chosen through member nomination and votes. 

## Technologies Used

### Frontend

- React
- Universal-Cookie
- React-router-dom
- Font Awesome

### Backend

- Node
- Express
- MongoDB
- Mongoose
- CORS
- bcrypt
- JSON Web Token
- Postman
